# censodemograficoueva2020.1
Programa criado a partir de uma avaliação da disciplina de primeiro período Lógica de Programação na Universidade Estadual Vale do Acaraú, Sobral, Ceará.
Cada funcionaliade funciona em partes (questão1, questão2...) e o programa para funcionar, depende da biblioteca censodemografico.py que tem sub-rotinas para ler e gravar arquivo
e tem a funcionalidade de cadastro.
